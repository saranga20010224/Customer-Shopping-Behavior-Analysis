# Customer-Shopping-Behavior-Analysis
End to End Data Analytics Project using Python, SQL Server, and Power BI

Customer Shopping Behavior Analytics Dashboard
Project Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, SQL Server, and Power BI. The objective was to analyze customer shopping behavior and generate business insights related to revenue, customer demographics, product performance, subscription patterns, and customer satisfaction.

Key Insights
Clothing generated the highest revenue.
Male customers contributed the largest share of sales.
Non-subscribers represented the majority of customers.
Customer ratings remained relatively consistent across product categories.
